# Info Cards Block

The Info Cards block displays a row of text-only cards with blue borders, typically used for highlighting company promises or core values.

## Authoring Pattern

Each row in the block table represents one card.

| Content                                                                    |
| -------------------------------------------------------------------------- |
| ## YOU ARE IN SAFE HANDS <br> We help to give you peace of mind...         |
| ## WE KEEP OUR PROMISES <br> We have paid over K1.6 billion...             |
| ## WE ARE HERE FOR YOU <br> From birth to the moment you leave a legacy... |

## Implementation Details

- **Layout**: Flexbox-based grid. 3 columns on desktop, 1 column on mobile.
- **Design**: `1.5px` blue border (`#28aae1`), `10px` rounded corners.
- **Typography**: Dark blue (`#00247d`) for both headings and text.
- **Micro-interactions**: Subtle hover lift effect with shadow.
