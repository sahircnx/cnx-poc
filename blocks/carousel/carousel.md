# Carousel Block

The Carousel block allows you to create a high-impact, full-width slider with background images and text overlays.

## Authoring Pattern

| Background Image                | Content                                                              |
| ------------------------------- | -------------------------------------------------------------------- |
| ![Slide 1](/path/to/image.jpg)  | # SLIDE TITLE <br> Slide description text. <br> [Button Label](link) |
| ![Slide 2](/path/to/image2.jpg) | # ANOTHER SLIDE <br> More description text. <br> [Explore](link)     |

## Implementation Details

- **Transitions**: Crossfade (1s duration).
- **Autoplay**: Enabled by default (6s interval). Pauses on mouse hover or when the slide is not visible in the viewport.
- **Navigation**: Dot indicators at the bottom center.
- **Design**: Left-aligned content with a subtle dark gradient for readability.
- **Typography**: Uses Roboto Black (900) for headings.
